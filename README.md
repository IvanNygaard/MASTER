# MASTER
Julia (SASQ) code for checking correction terms of the cl-CCSD-eom Hamiltonian and Python code for visualization of data. Data obtained by running cl-CCSD and cl-CCSD-eom calculations with the eT program.

The modifications to the coupled cluster code in the eT program were carried out a on the development branch hf-in-proper-localization (on GitLab).
